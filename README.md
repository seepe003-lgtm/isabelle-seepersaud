# Isabelle Seepersaud - Personal Website

This is a personal website showcasing my projects, hobbies, career information, and resume.

## Website Structure

- `index.html` - Home page
- `hobbies.html` - Hobbies and interests
- `discover.html` - Discover UMD page
- `resume.html` - Resume and qualifications
- `career.html` - Career information
- `CSS/` - Stylesheet files
- `media/` - Images, videos, and audio files
- `js/` - JavaScript files
- `files/` - Downloadable files (e.g., PDF resume)

## GitHub Pages Deployment

This website is automatically deployed to GitHub Pages using GitHub Actions.

The deployment workflow is triggered on:
- Push to the `main` branch
- Manual workflow dispatch

### Setting Up GitHub Pages

1. Go to your repository settings
2. Navigate to "Pages" in the left sidebar
3. Under "Source", select "GitHub Actions"
4. The workflow will automatically deploy the site when changes are pushed to main

The website will be available at: `https://seepe003-lgtm.github.io/isabelle-seepersaud/`

## Local Development

To view the website locally:
1. Clone the repository
2. Open `index.html` in your web browser
3. Or use a local web server (e.g., `python -m http.server`)
